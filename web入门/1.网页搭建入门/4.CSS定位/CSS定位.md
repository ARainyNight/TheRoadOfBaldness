##CSS-position

### 课程目标

- 学习CSS中的定位模型
- 使用定位模型来达到布局目的
- 学习position
  - `static` : `自然模型`
  - `relative` :`相对定位模型`
  - `absolute` :`绝对定位模型`
  - `fixed` :`固定定位模型`
  - `sticky` :`磁贴定位模型`
