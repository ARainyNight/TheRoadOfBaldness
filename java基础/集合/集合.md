## 集合类

### 为什么会出现集合类

- 面向对象语言对事物的体现都是以对象的形式，所以为了方便对多个对象的操作，就对对象进行存储，集合就是存储对象最常用的一种方式

### 数组和集合类同是容器，有何不同？

+ 数组虽然也可以存储对象，但长度是固定的；集合长度是可变的
+ 数组中可以存储基本数据类型，集合只能存储对象

### 集合类的特点

+ 集合只用于存储对象
+ 集合长度是可变的
+ 集合可以储存不同类型的对象
